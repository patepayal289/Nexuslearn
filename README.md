# 🎓 NexusLearn

NexusLearn is a modern online learning platform designed to help students discover, access, and manage educational courses in a simple and engaging way.

## 🚀 Live Demo

🌐 https://nexuslearn-eta.vercel.app/

## ✨ Features

* Modern and responsive UI
* Course browsing and discovery
* Interactive learning experience
* Mobile-friendly design
* Fast performance
* Clean and intuitive navigation
* Built with modern web technologies

## 🛠️ Tech Stack

* Next.js
* React.js
* TypeScript
* Tailwind CSS
* Vercel Deployment

## 📂 Installation

Clone the repository:

```bash
git clone https://github.com/your-username/nexuslearn.git
```

Navigate to the project folder:

```bash
cd nexuslearn
```

Install dependencies:

```bash
npm install
```

Run the development server:

```bash
npm run dev
```

Open:

```bash
http://localhost:3000
```
